üìç Komponen Utama

1. Cell (Sel)

Kotak terkecil tempat data diinput. Setiap cell memiliki alamat unik (contoh: A1, B2, C3).

2. Row (Baris)

Deretan horizontal cell, diberi nomor 1, 2, 3, dst. Excel/Sheets punya jutaan baris.

3. Column (Kolom)

Deretan vertikal cell, diberi huruf A, B, C, ..., Z, AA, AB, dst. Total 16.384 kolom di Excel.

4. Range

Kumpulan beberapa cell. Contoh: A1:C10 (dari A1 sampai C10).

5. Formula Bar

Tempat mengetik atau melihat isi cell/rumus yang sedang dipilih.

6. Sheet/Tab

Satu file bisa punya banyak sheet (lembar kerja). Berguna untuk organisasi data.
