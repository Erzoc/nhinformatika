üìö Bagian 1: Pengenalan Data & Spreadsheet

üî¢ Jenis-Jenis Data

1. Data Kualitatif

Data deskriptif berupa teks/huruf yang menggambarkan kualitas.

Contoh: Nama, Warna, Jenis Kelamin, Status

2. Data Kuantitatif

Data numerik (angka) yang dapat diukur dan dihitung.

Diskrit: Angka bulat (30 siswa)

Kontinu: Angka desimal (165.5 cm)

üíª Excel vs Google Sheets

üìä Microsoft Excel

  • ‚úì Fitur sangat lengkap & advanced
  • ‚úì Performa cepat untuk data besar
  • ‚úì Macro & VBA programming
  • ‚úì Dapat bekerja offline
  • ‚úó Berbayar (kecuali Excel Online)

üìà Google Sheets

  • ‚úì Gratis 100% (butuh akun Google)
  • ‚úì Kolaborasi real-time
  • ‚úì Auto-save ke cloud
  • ‚úì Akses dari perangkat apa saja
  • ‚úó Perlu internet untuk akses penuh
üí° Kesimpulan: 90% rumus Excel juga berfungsi di Google Sheets! Pilih Excel untuk analisis kompleks & data besar, pilih Sheets untuk kolaborasi & akses mobile.

üñ•Ô∏è Komponen Penting Spreadsheet

  • Cell (Sel): Kotak terkecil dengan alamat unik (A1, B2, C3)
  • Row (Baris): Deretan horizontal, diberi nomor 1, 2, 3...
  • Column (Kolom): Deretan vertikal, diberi huruf A, B, C...
  • Range: Kumpulan cell (A1:C10)
  • Formula Bar: Tempat mengetik/melihat isi cell
  • Sheet/Tab: Lembar kerja dalam satu file

üî¢ Bagian 2: Rumus & Fungsi Dasar

‚ûï Fungsi Matematika

SUM - Menjumlahkan

=SUM(A1:A10)

Menjumlahkan semua angka dalam range

AVERAGE - Rata-rata

=AVERAGE(B1:B10)

Menghitung nilai rata-rata

COUNT - Menghitung

=COUNT(C1:C10)

Hitung cell berisi angka

MAX & MIN

=MAX(D1:D10)
=MIN(D1:D10)

Nilai tertinggi & terendah

üîÄ Fungsi Logika

IF - Logika Percabangan

=IF(A1>=75; "LULUS"; "GAGAL")

Jika A1 ‚â• 75 maka "LULUS", jika tidak "GAGAL"

IF Bertingkat (Nested)

=IF(A1>=90;"A";IF(A1>=80;"B";"C"))

Grade A, B, atau C berdasarkan nilai

AND - Semua Harus Benar

=IF(AND(A1>=75;B1>=75);"LULUS";"GAGAL")

KEDUA nilai harus ‚â•75

OR - Salah Satu Benar

=IF(OR(A1="A";A1="B");"Baik";"Kurang")

Baik jika grade A ATAU B

üîç Fungsi Lookup

VLOOKUP - Pencarian Vertikal

=VLOOKUP(nilai_cari; tabel_data; nomor_kolom; FALSE)

Contoh: Mencari harga barang berdasarkan kode produk

=VLOOKUP("A02"; A1:B10; 2; FALSE)

Cari "A02" di kolom pertama, ambil nilai dari kolom ke-2

üí° Tips VLOOKUP

  • ‚Ä¢ Data harus diurutkan dari atas ke bawah
  • ‚Ä¢ Nilai yang dicari harus ada di kolom paling kiri
  • ‚Ä¢ Gunakan FALSE untuk exact match
  • ‚Ä¢ Nomor kolom dimulai dari 1 (bukan 0)

üìù Fungsi Teks

UPPER/LOWER/PROPER

=UPPER("hello") ‚Üí HELLO
=LOWER("HELLO") ‚Üí hello
=PROPER("hello world") ‚Üí Hello World

CONCAT - Gabung Teks

=CONCAT(A1;" ";B1)

Menggabungkan isi A1 dan B1 dengan spasi

LEFT/RIGHT - Ambil Karakter

=LEFT("Excel";2) ‚Üí Ex
=RIGHT("Excel";3) ‚Üí cel

üìä Bagian 3: Visualisasi Data & Grafik

üìà Jenis-Jenis Chart

1. Column Chart üìä

Kapan: Membandingkan data antar kategori

Contoh: Penjualan per bulan, nilai per kelas

2. Line Chart üìà

Kapan: Tren data sepanjang waktu

Contoh: Harga saham, suhu harian

3. Pie Chart ü•ß

Kapan: Proporsi/persentase

Contoh: Market share, komposisi budget

‚ö†Ô∏è Maksimal 5-7 kategori!

4. Bar Chart (Horizontal)

Kapan: Ranking dengan label panjang

Contoh: Perbandingan negara

5. Scatter Plot ‚ö´

Kapan: Hubungan 2 variabel

Contoh: Tinggi-berat, jam belajar-nilai

6. Area Chart

Kapan: Volume/akumulasi over time

Contoh: Total penjualan kumulatif

üõ†Ô∏è Langkah Membuat Chart

  1. 1. Persiapkan Data: Susun rapi, header di baris pertama, tidak ada cell kosong
  2. 2. Pilih Data: Blok seluruh data termasuk header
  3. 3. Insert Chart: Excel ‚Üí Insert ‚Üí Charts | Sheets ‚Üí Insert ‚Üí Chart
  4. 4. Customize: Tambahkan title, label axis, atur warna, hapus elemen tidak perlu

üé® Tips Desain Chart Efektif

  • ‚úì Simplicity: Less is more - fokus pada message
  • ‚úì Warna: Gunakan palet konsisten, highlight data penting
  • ‚úì Label: Title jelas, axis labels wajib ada
  • ‚úì Context: Sertakan sumber data & periode waktu
  • ‚úó Hindari: 3D chart, rainbow colors, data overload

üîÑ Pivot Table

Definisi: Alat untuk meringkas dan menganalisis data besar dengan cara fleksibel tanpa mengubah data asli.

‚úÖ Manfaat Pivot Table

  • ‚Ä¢ Summarize data besar dengan cepat
  • ‚Ä¢ Grouping & filtering dinamis
  • ‚Ä¢ Kalkulasi otomatis (sum, avg, count)
  • ‚Ä¢ Analisis multi-dimensional
  • ‚Ä¢ Drag-and-drop interface

Cara Membuat:

  1. 1. Pilih data ‚Üí Insert ‚Üí PivotTable (Excel) / Pivot table (Sheets)
  2. 2. Drag field ke area: Rows (baris), Columns (kolom), Values (nilai), Filters
  3. 3. Pilih fungsi aggregasi: SUM, AVERAGE, COUNT, MAX, MIN
  4. 4. Customize format dan layout sesuai kebutuhan

üìä Dashboard

Definisi: Tampilan visual yang menampilkan metrics dan KPI penting dalam satu layar untuk monitoring dan decision making cepat.

Elemen Dashboard

  • ‚Ä¢ KPI Cards (angka besar metric utama)
  • ‚Ä¢ Trend Charts (line/area)
  • ‚Ä¢ Comparison Charts (bar/column)
  • ‚Ä¢ Composition (pie/donut)
  • ‚Ä¢ Filters/Slicers (dropdown)
  • ‚Ä¢ Tables (detail data)

Prinsip Desain Dashboard

  • ‚Ä¢ Prioritas: Info penting di kiri atas
  • ‚Ä¢ Konsistensi: Warna & font seragam
  • ‚Ä¢ White space: Beri ruang bernafas
  • ‚Ä¢ Actionable: Bantu decision making
  • ‚Ä¢ Real-time: Update otomatis
  • ‚Ä¢ Mobile-friendly: Responsif

üéØ Key Takeaways (Poin Penting)

üí° 10 Hal yang Harus Dikuasai

  1. Bedakan data kualitatif vs kuantitatif untuk pemilihan analisis yang tepat
  2. Pahami interface spreadsheet - cell, row, column, range, formula bar
  3. Kuasai rumus matematika dasar: SUM, AVERAGE, COUNT, MAX, MIN
  4. Gunakan IF untuk logika percabangan dan kombinasikan dengan AND/OR
  5. VLOOKUP adalah senjata ampuh untuk mencari data dari tabel master
  6. Pilih jenis chart yang tepat sesuai tujuan komunikasi data
  7. Desain chart yang efektif: simple, jelas, fokus pada message
  8. Pivot Table untuk analisis cepat data besar tanpa rumus kompleks
  9. Dashboard harus actionable dan membantu decision making
  10. Praktik terus-menerus adalah kunci menguasai Excel/Sheets!
