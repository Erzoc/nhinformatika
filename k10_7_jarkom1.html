Jaringan Komputer

Memahami konsep, pengelompokan, dan implementasi jaringan komputer dari LAN hingga Internet untuk era digital

üìö Pengertian Jaringan Komputer

Apa itu Jaringan Komputer?

Jaringan komputer (Computer Network) adalah dua atau lebih komputer yang saling terhubung untuk berbagi sumber daya (resource sharing), berkomunikasi, dan bertukar data/informasi.

üéØ Tujuan Jaringan Komputer

  • ‚Ä¢ Resource Sharing: Berbagi printer, file, storage, aplikasi
  • ‚Ä¢ Komunikasi: Email, chat, video conference
  • ‚Ä¢ Akses Informasi: Internet, database, server
  • ‚Ä¢ Efisiensi: Satu printer untuk banyak komputer
  • ‚Ä¢ Kolaborasi: Kerja tim secara online
  • ‚Ä¢ Backup & Recovery: Data tersimpan di server pusat

‚úÖ Manfaat Jaringan Komputer

  • ‚Ä¢ Hemat biaya: Tidak perlu beli hardware untuk tiap komputer
  • ‚Ä¢ Fast communication: Transfer data cepat & real-time
  • ‚Ä¢ Centralized data: Data terpusat, mudah dikelola
  • ‚Ä¢ Remote access: Akses dari mana saja (internet)
  • ‚Ä¢ Security: Kontrol akses terpusat
  • ‚Ä¢ Scalability: Mudah tambah komputer baru

üí° Komponen Jaringan Komputer

  • ‚Ä¢ Hardware: Komputer, server, router, switch, kabel, wireless
  • ‚Ä¢ Software: Sistem operasi jaringan, protokol (TCP/IP)
  • ‚Ä¢ Media transmisi: Kabel (UTP, fiber optik) atau wireless (Wi-Fi, 4G/5G)
  • ‚Ä¢ Protokol: Aturan komunikasi (HTTP, FTP, TCP/IP)

üóÇÔ∏è Pengelompokan Jaringan Komputer

Klasifikasi Berdasarkan Area/Jangkauan

Jaringan komputer dikelompokkan berdasarkan jangkauan geografis dari yang paling kecil hingga global.

1. PAN (Personal Area Network)

  • ‚Ä¢ Jangkauan: Sangat kecil (0-10 meter)
  • ‚Ä¢ Contoh: Bluetooth, USB, Infrared
  • ‚Ä¢ Penggunaan: Hubungkan HP ke laptop, wireless mouse, headset Bluetooth
  • ‚Ä¢ Kelebihan: Praktis, portable, mudah setup
  • ‚Ä¢ Kekurangan: Jangkauan sangat terbatas

2. LAN (Local Area Network)

  • ‚Ä¢ Jangkauan: Kecil (10-1000 meter)
  • ‚Ä¢ Area: Satu gedung, kantor, sekolah, rumah
  • ‚Ä¢ Contoh: Jaringan lab komputer, kantor
  • ‚Ä¢ Media: Kabel UTP, Wi-Fi
  • ‚Ä¢ Kecepatan: Tinggi (100 Mbps - 10 Gbps)
  • ‚Ä¢ Biaya: Murah, mudah maintenance

3. MAN (Metropolitan Area Network)

  • ‚Ä¢ Jangkauan: Sedang (1-50 km)
  • ‚Ä¢ Area: Satu kota/metropolitan
  • ‚Ä¢ Contoh: Jaringan antar kampus dalam satu kota, jaringan bank cabang
  • ‚Ä¢ Media: Fiber optik, wireless
  • ‚Ä¢ Kecepatan: Tinggi
  • ‚Ä¢ Biaya: Lebih mahal dari LAN

4. WAN (Wide Area Network)

  • ‚Ä¢ Jangkauan: Luas (>50 km, antar kota/negara)
  • ‚Ä¢ Area: Global, antar benua
  • ‚Ä¢ Contoh: Internet, jaringan perusahaan multinasional
  • ‚Ä¢ Media: Fiber optik, satelit, leased line
  • ‚Ä¢ Kecepatan: Bervariasi
  • ‚Ä¢ Biaya: Sangat mahal

üí° Perbandingan Singkat

  • ‚Ä¢ PAN: Personal (HP ‚Üî Laptop via Bluetooth)
  • ‚Ä¢ LAN: Local (Satu gedung/kantor/sekolah)
  • ‚Ä¢ MAN: Metropolitan (Satu kota)
  • ‚Ä¢ WAN: Wide (Antar kota/negara/benua = Internet)

Pengelompokan Jaringan Lainnya

Berdasarkan Topologi

  • ‚Ä¢ Star: Semua komputer terhubung ke switch/hub pusat
  • ‚Ä¢ Bus: Satu kabel backbone, semua komputer terhubung ke sana
  • ‚Ä¢ Ring: Bentuk lingkaran, data berputar
  • ‚Ä¢ Mesh: Setiap komputer terhubung ke semua komputer lain
  • ‚Ä¢ Hybrid: Kombinasi beberapa topologi

Berdasarkan Fungsi

  • ‚Ä¢ Client-Server: Ada server pusat, client request ke server
  • ‚Ä¢ Peer-to-Peer (P2P): Semua komputer setara, tidak ada server khusus

Berdasarkan Media Transmisi

  • ‚Ä¢ Wired (Kabel): UTP, Coaxial, Fiber Optik
  • ‚Ä¢ Wireless (Nirkabel): Wi-Fi, Bluetooth, 4G/5G, Satelit

Berdasarkan Akses

  • ‚Ä¢ Intranet: Jaringan internal perusahaan, privat
  • ‚Ä¢ Extranet: Intranet yang bisa diakses partner tertentu
  • ‚Ä¢ Internet: Jaringan global, publik

üè¢ Jaringan Lokal (LAN) - Local Area Network

Pendalaman LAN

LAN (Local Area Network) adalah jaringan komputer dalam area terbatas seperti rumah, kantor, sekolah, atau gedung. LAN adalah tipe jaringan yang paling umum digunakan.

Karakteristik LAN

  • ‚Ä¢ Area terbatas: Satu lokasi fisik
  • ‚Ä¢ Kecepatan tinggi: 100 Mbps - 10 Gbps
  • ‚Ä¢ Biaya rendah: Murah untuk dibangun & maintain
  • ‚Ä¢ Ownership: Dimiliki oleh satu organisasi
  • ‚Ä¢ Latency rendah: Delay minimal
  • ‚Ä¢ Error rate rendah: Transmisi data akurat

Perangkat LAN

  • ‚Ä¢ Komputer/Laptop: End devices (client)
  • ‚Ä¢ Switch: Menghubungkan banyak komputer
  • ‚Ä¢ Router: Menghubungkan LAN ke internet
  • ‚Ä¢ Access Point (AP): Wi-Fi untuk wireless
  • ‚Ä¢ Kabel UTP: Cat5e, Cat6 untuk koneksi
  • ‚Ä¢ Server: Menyimpan data & aplikasi

Topologi LAN Umum

  • ‚Ä¢ Star Topology: Paling populer, semua ke switch pusat
  • ‚Ä¢ Extended Star: Star bertingkat dengan beberapa switch
  • ‚Ä¢ Wireless (Wi-Fi): Access Point sebagai pusat

Contoh Implementasi LAN

  • ‚Ä¢ Lab komputer sekolah: 30 komputer ‚Üí 1 switch ‚Üí 1 router ‚Üí internet
  • ‚Ä¢ Kantor: Semua karyawan sharing printer, file server
  • ‚Ä¢ Rumah (Home Network): WiFi router untuk semua device
  • ‚Ä¢ Warnet: Banyak komputer ‚Üí switch ‚Üí billing server

‚úÖ Keuntungan LAN

  • ‚Ä¢ Resource sharing: Satu printer untuk semua komputer
  • ‚Ä¢ File sharing: Transfer file cepat antar komputer
  • ‚Ä¢ Centralized data: Data di server, mudah backup
  • ‚Ä¢ Internet sharing: Satu koneksi internet untuk banyak device
  • ‚Ä¢ Game multiplayer: Main game LAN tanpa internet
  • ‚Ä¢ Communication: Chat, video call internal

‚ö†Ô∏è Kekurangan LAN

  • ‚Ä¢ Area terbatas: Hanya dalam satu lokasi
  • ‚Ä¢ Security risk: Jika satu komputer kena virus, bisa menyebar
  • ‚Ä¢ Maintenance: Perlu IT admin untuk kelola
  • ‚Ä¢ Initial cost: Biaya awal beli switch, kabel, instalasi

üåç Internet - Jaringan Global

Internet: The World Wide Network

Internet adalah jaringan komputer global yang menghubungkan jutaan jaringan di seluruh dunia. Internet adalah bentuk WAN (Wide Area Network) terbesar dan paling kompleks.

üåê Fakta Internet

  • ‚Ä¢ Pengguna global: 5.4+ miliar orang (2024)
  • ‚Ä¢ Website: 1.9+ miliar website aktif
  • ‚Ä¢ Data per hari: 328.77 juta terabyte
  • ‚Ä¢ Kecepatan rata-rata: Bervariasi 1 Mbps - 1 Gbps+
  • ‚Ä¢ Lahir: ARPANET (1969) ‚Üí Internet komersial (1990s)

Cara Kerja Internet

  • ‚Ä¢ Protokol TCP/IP: Aturan komunikasi standar
  • ‚Ä¢ IP Address: Alamat unik setiap device (192.168.1.1)
  • ‚Ä¢ DNS: Mengubah domain (google.com) jadi IP address
  • ‚Ä¢ Packet switching: Data dipecah jadi paket-paket kecil
  • ‚Ä¢ Routing: Paket mencari jalur terbaik ke tujuan
  • ‚Ä¢ ISP: Internet Service Provider (Telkom, Indihome, dll)

Infrastruktur Internet

  • ‚Ä¢ Backbone: Kabel fiber optik bawah laut
  • ‚Ä¢ Data Center: Server farm raksasa (Google, Amazon)
  • ‚Ä¢ IXP: Internet Exchange Point, tempat ISP bertemu
  • ‚Ä¢ Satelit: Untuk area terpencil
  • ‚Ä¢ Tower BTS: Untuk internet mobile (4G/5G)

Layanan Internet

  • ‚Ä¢ WWW (Web): Website, browser (Chrome, Firefox)
  • ‚Ä¢ Email: Gmail, Outlook, komunikasi elektronik
  • ‚Ä¢ Social Media: Facebook, Instagram, Twitter, TikTok
  • ‚Ä¢ Streaming: YouTube, Netflix, Spotify
  • ‚Ä¢ Cloud Storage: Google Drive, Dropbox, OneDrive
  • ‚Ä¢ E-commerce: Tokopedia, Shopee, Amazon
  • ‚Ä¢ Gaming: Online multiplayer games

Jenis Koneksi Internet

  • ‚Ä¢ DSL: Melalui kabel telepon
  • ‚Ä¢ Cable: Melalui kabel TV (coaxial)
  • ‚Ä¢ Fiber Optik: Kecepatan super tinggi (IndiHome Fiber)
  • ‚Ä¢ Wireless: 4G/5G, Wi-Fi publik
  • ‚Ä¢ Satelit: Untuk daerah terpencil (Starlink)

‚úÖ Manfaat Internet

  • ‚Ä¢ Informasi unlimited: Akses pengetahuan global
  • ‚Ä¢ Komunikasi instan: Email, chat, video call worldwide
  • ‚Ä¢ E-commerce: Belanja online dari mana saja
  • ‚Ä¢ Entertainment: Streaming video, musik, game
  • ‚Ä¢ E-learning: Belajar online (Coursera, YouTube)
  • ‚Ä¢ Remote work: Kerja dari rumah
  • ‚Ä¢ IoT: Smart home, smart city

‚ö†Ô∏è Risiko & Tantangan Internet

  • ‚Ä¢ Privacy: Data pribadi bisa bocor
  • ‚Ä¢ Cybercrime: Hacking, phishing, malware
  • ‚Ä¢ Hoax: Informasi palsu menyebar cepat
  • ‚Ä¢ Addiction: Kecanduan internet & media sosial
  • ‚Ä¢ Digital divide: Kesenjangan akses internet
  • ‚Ä¢ Bandwidth: Kecepatan tergantung infrastruktur

üí° Tips Aman Berinternet

  • ‚Ä¢ Password kuat: Kombinasi huruf, angka, simbol
  • ‚Ä¢ 2FA: Two-Factor Authentication untuk keamanan extra
  • ‚Ä¢ HTTPS: Cek gembok hijau saat browsing
  • ‚Ä¢ Antivirus: Install dan update rutin
  • ‚Ä¢ Privacy setting: Atur siapa bisa lihat data Anda
  • ‚Ä¢ Think before post: Jangan share data sensitif
  • ‚Ä¢ VPN: Untuk enkripsi koneksi di WiFi publik

üîß Perangkat Jaringan Komputer

Hardware untuk Membangun Jaringan

1. NIC (Network Interface Card)

  • ‚Ä¢ Fungsi: Kartu jaringan untuk koneksi komputer ke jaringan
  • ‚Ä¢ Jenis: Ethernet (LAN port), Wi-Fi adapter
  • ‚Ä¢ MAC Address: Alamat fisik unik setiap NIC

2. Switch

  • ‚Ä¢ Fungsi: Menghubungkan banyak komputer dalam LAN
  • ‚Ä¢ Smart device: Bisa filter & forward data ke tujuan
  • ‚Ä¢ Port: 8, 16, 24, 48 port
  • ‚Ä¢ Kecepatan: 10/100/1000 Mbps (Gigabit)

3. Router

  • ‚Ä¢ Fungsi: Menghubungkan 2 jaringan berbeda (LAN ke WAN/Internet)
  • ‚Ä¢ Routing: Mencari jalur terbaik untuk data
  • ‚Ä¢ NAT: Translate IP private ke public
  • ‚Ä¢ Firewall: Keamanan jaringan

4. Access Point (AP)

  • ‚Ä¢ Fungsi: Menyediakan koneksi Wi-Fi wireless
  • ‚Ä¢ Mode: Access Point, Repeater, Bridge
  • ‚Ä¢ Coverage: Radius 50-100 meter indoor

5. Modem

  • ‚Ä¢ Fungsi: Modulator-Demodulator, ubah sinyal digital ‚Üî analog
  • ‚Ä¢ Jenis: DSL modem, Cable modem, 4G/5G modem
  • ‚Ä¢ ISP: Dari provider internet (Telkom, dll)

6. Kabel & Media Transmisi

  • ‚Ä¢ UTP (Unshielded Twisted Pair): Cat5e, Cat6, RJ45 connector
  • ‚Ä¢ Fiber Optik: Kecepatan sangat tinggi, jarak jauh
  • ‚Ä¢ Coaxial: Untuk kabel TV & internet
  • ‚Ä¢ Wireless: Wi-Fi (2.4 GHz, 5 GHz), Bluetooth

üí° Perbedaan Switch vs Hub vs Router

  • ‚Ä¢ Hub: Broadcast ke semua port (dumb device, jarang dipakai)
  • ‚Ä¢ Switch: Forward ke port tujuan saja (smart, paling umum di LAN)
  • ‚Ä¢ Router: Hubungkan jaringan berbeda, routing antar network
